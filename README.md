# PopX Mobile App

A modern, pixel-perfect mobile application built with React and Tailwind CSS, featuring seamless navigation and a beautiful user interface.

**🚀 View Live:** [PopX Mobile App on Vercel](https://your-vercel-url.vercel.app)

## 🚀 Features

- **📱 Mobile-First Design**: Optimized for mobile devices with centered layout (375x812px)
- **🎨 Pixel-Perfect UI**: Exact match to design specifications with proper spacing
- **🔄 Smooth Navigation**: Intuitive page transitions with bottom pagination
- **💫 Modern Animations**: Smooth transitions and micro-interactions
- **🎯 Interactive Elements**: Hover states, focus effects, and touch-optimized controls
- **📋 Form Validation**: Proper input types and validation patterns
- **🌈 Custom Branding**: Purple accent theme (#6C25FF) with consistent design language
- **⚡ Performance Optimized**: Fast loading with optimized build sizes (77.31 kB JS, 2.78 kB CSS)

## 📱 Pages

- **🏠 Home**: Landing page with navigation to signup/login
- **🔐 Login**: User authentication interface with floating inputs
- **📝 Signup**: Comprehensive registration form with validation
- **👤 Profile**: User account settings with avatar and camera icon

## 🛠 Tech Stack

- **⚛️ React 19**: Modern React with latest features and hooks
- **🚀 React Router DOM**: Client-side routing with smooth transitions
- **🎨 Tailwind CSS**: Utility-first CSS framework with custom configuration
- **🎭 Lucide React**: Beautiful icon library for consistent UI
- **📦 PostCSS**: CSS processing and optimization
- **🌐 Vercel**: Production deployment with global CDN

## 🎨 Design Specifications

- **📐 Mobile Dimensions**: 375x812px (iPhone X/11/12 dimensions)
- **🎨 Color Scheme**: 
  - Primary: `#6C25FF` (Purple accent)
  - Background: `#f7f8f9` (Light gray)
  - Text: `#1D2226` (Dark gray)
- **🔤 Typography**: Rubik font family for modern, readable text
- **📱 Responsive**: Centered layout that works on all screen sizes
- **🎯 Touch Optimized**: Proper touch targets and mobile interactions

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/amit9145/Intern-Project.git

# Navigate to project directory
cd PopX-Mobile-App

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Deploy to Vercel
npm run deploy
```
